# Aether IDE
Aether IDE : Un éditeur de code léger et performant construit avec Tauri (React + Rust). Équipé de Monaco Editor, support natif pour les fichiers, dialogues et intégrations système. Idéal pour le développement cross-platform (Windows, macOS, Linux).  🚀 npm run tauri dev pour démarrer !  #Tauri #React #Rust #IDE
